# Quick Start

## Project Initialization

To create and initialize a new Gez project, follow these steps:

```bash
mkdir your-app-name
cd your-app-name
```

## Framework Version Selection

The Gez framework offers multiple technology stack versions, each optimized for different application scenarios. Choose the appropriate version based on your project requirements:

### HTML

Ideal for projects pursuing a minimalist architecture:

- Zero external dependencies, ready to use out of the box
- Native JavaScript development experience
- Suitable for building static websites and lightweight applications
- Supports progressive feature expansion

[View HTML Version Documentation](../frameworks/html.md)

### Vue2

Designed for enterprise-level application development:

- Full TypeScript support
- Rich ecosystem of third-party components
- Mature development toolchain
- Proven stability in production environments

[View Vue2 Version Documentation](../frameworks/vue2.md)

### Vue3

Tailored for modern web application development:

- Proxy-based reactivity system
- Composition API support
- Improved runtime performance
- Smaller bundle size

[View Vue3 Version Documentation](../frameworks/vue3.md)

### Preact+HTM

Perfect for projects prioritizing lightweight and high performance:

- Minimal runtime size (3KB)
- Native JavaScript template syntax
- React-compatible API
- Excellent performance

[View Preact+HTM Version Documentation](../frameworks/preact-htm.md)

## AI-Assisted Development

The Gez framework provides AI-assisted development capabilities to significantly enhance development efficiency:

1. Select the appropriate framework version documentation
2. Provide the documentation content to the AI assistant
3. The AI will automatically generate the project skeleton and configuration files

:::tip Tip
AI-assisted development not only speeds up project initialization but also ensures that the project structure adheres to best practices.
:::
