# 快速開始

## 專案初始化

建立並初始化一個新的 Gez 專案，請按照以下步驟操作：

```bash
mkdir your-app-name
cd your-app-name
```

## 框架版本選擇

Gez 框架提供了多個技術棧版本，每個版本都針對不同的應用場景進行了優化。請根據專案需求選擇合適的版本：

### HTML

適用於追求極簡架構的專案：

- 零外部依賴，啟動即用
- 原生 JavaScript 開發體驗
- 適合建置靜態網站和輕量級應用
- 支援漸進式功能擴展

[查看 HTML 版本詳細文件](../frameworks/html.md)

### Vue2

適用於企業級應用開發：

- 完整的 TypeScript 支援
- 豐富的第三方元件生態
- 成熟的開發工具鏈
- 經過生產環境驗證的穩定性

[查看 Vue2 版本詳細文件](../frameworks/vue2.md)

### Vue3

適用於現代化 Web 應用開發：

- 基於 Proxy 的響應式系統
- 組合式 API 支援
- 更優的執行時效能
- 更小的打包體積

[查看 Vue3 版本詳細文件](../frameworks/vue3.md)

### Preact+HTM

適用於追求輕量級和高效能的專案：

- 極小的執行時體積（3KB）
- 原生 JavaScript 模板語法
- 與 React 相容的 API
- 優秀的效能表現

[查看 Preact+HTM 版本詳細文件](../frameworks/preact-htm.md)

## AI 輔助開發

Gez 框架提供了 AI 輔助開發能力，可以顯著提升開發效率：

1. 選擇適合的框架版本文檔
2. 將文件內容提供給 AI 助手
3. AI 將自動生成專案骨架和設定檔

:::tip 提示
AI 輔助開發不僅能加快專案初始化速度，還能確保專案結構符合最佳實踐。
:::
