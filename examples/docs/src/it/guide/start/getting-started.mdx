# Guida Rapida

## Inizializzazione del Progetto

Per creare e inizializzare un nuovo progetto Gez, segui questi passaggi:

```bash
mkdir nome-della-tua-app
cd nome-della-tua-app
```

## Scelta della Versione del Framework

Il framework Gez offre diverse versioni per stack tecnologici, ciascuna ottimizzata per diversi scenari di applicazione. Scegli la versione più adatta in base alle esigenze del tuo progetto:

### HTML

Ideale per progetti che richiedono un'architettura minimalista:

- Nessuna dipendenza esterna, pronto all'uso
- Esperienza di sviluppo con JavaScript nativo
- Adatto per la costruzione di siti web statici e applicazioni leggere
- Supporto per l'estensione progressiva delle funzionalità

[Consulta la documentazione dettagliata per la versione HTML](../frameworks/html.md)

### Vue2

Adatto per lo sviluppo di applicazioni aziendali:

- Supporto completo per TypeScript
- Ampio ecosistema di componenti di terze parti
- Strumenti di sviluppo maturi
- Stabilità verificata in ambienti di produzione

[Consulta la documentazione dettagliata per la versione Vue2](../frameworks/vue2.md)

### Vue3

Adatto per lo sviluppo di applicazioni web moderne:

- Sistema reattivo basato su Proxy
- Supporto per API composizionali
- Migliori prestazioni a runtime
- Volume di pacchetto ridotto

[Consulta la documentazione dettagliata per la versione Vue3](../frameworks/vue3.md)

### Preact+HTM

Adatto per progetti che richiedono leggerezza e alte prestazioni:

- Volume di runtime estremamente ridotto (3KB)
- Sintassi di template JavaScript nativa
- API compatibile con React
- Eccellenti prestazioni

[Consulta la documentazione dettagliata per la versione Preact+HTM](../frameworks/preact-htm.md)

## Sviluppo Assistito da AI

Il framework Gez offre capacità di sviluppo assistito da AI, che possono migliorare significativamente l'efficienza dello sviluppo:

1. Seleziona la documentazione della versione del framework più adatta
2. Fornisci il contenuto della documentazione all'assistente AI
3. L'AI genererà automaticamente lo scheletro del progetto e i file di configurazione

:::tip Suggerimento
Lo sviluppo assistito da AI non solo accelera l'inizializzazione del progetto, ma garantisce anche che la struttura del progetto sia conforme alle migliori pratiche.
:::
