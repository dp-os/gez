# Démarrage rapide

## Initialisation du projet

Pour créer et initialiser un nouveau projet Gez, suivez les étapes suivantes :

```bash
mkdir votre-nom-d-application
cd votre-nom-d-application
```

## Choix de la version du framework

Le framework Gez propose plusieurs versions de piles technologiques, chacune optimisée pour différents cas d'utilisation. Choisissez la version adaptée à vos besoins de projet :

### HTML

Idéal pour les projets recherchant une architecture minimaliste :

- Aucune dépendance externe, prêt à l'emploi
- Expérience de développement en JavaScript natif
- Convient pour la construction de sites web statiques et d'applications légères
- Supporte l'extension progressive des fonctionnalités

[Consulter la documentation détaillée de la version HTML](../frameworks/html.md)

### Vue2

Idéal pour le développement d'applications d'entreprise :

- Support complet de TypeScript
- Écosystème riche de composants tiers
- Chaîne d'outils de développement mature
- Stabilité éprouvée en environnement de production

[Consulter la documentation détaillée de la version Vue2](../frameworks/vue2.md)

### Vue3

Idéal pour le développement d'applications web modernes :

- Système réactif basé sur Proxy
- Support de l'API de composition
- Meilleures performances d'exécution
- Volume de bundle plus réduit

[Consulter la documentation détaillée de la version Vue3](../frameworks/vue3.md)

### Preact+HTM

Idéal pour les projets recherchant légèreté et haute performance :

- Volume d'exécution extrêmement réduit (3KB)
- Syntaxe de template en JavaScript natif
- API compatible avec React
- Excellentes performances

[Consulter la documentation détaillée de la version Preact+HTM](../frameworks/preact-htm.md)

## Développement assisté par IA

Le framework Gez offre des capacités de développement assisté par IA, permettant d'améliorer significativement l'efficacité du développement :

1. Sélectionnez la documentation de la version du framework adaptée
2. Fournissez le contenu de la documentation à l'assistant IA
3. L'IA générera automatiquement le squelette du projet et les fichiers de configuration

:::tip Astuce
Le développement assisté par IA permet non seulement d'accélérer l'initialisation du projet, mais aussi de garantir que la structure du projet respecte les meilleures pratiques.
:::
