# Hızlı Başlangıç

## Proje Başlatma

Yeni bir Gez projesi oluşturmak ve başlatmak için aşağıdaki adımları izleyin:

```bash
mkdir your-app-name
cd your-app-name
```

## Framework Sürüm Seçimi

Gez Framework, farklı uygulama senaryoları için optimize edilmiş birden fazla teknoloji yığını sürümü sunar. Proje gereksinimlerinize göre uygun sürümü seçin:

### HTML

Minimalist mimariyi tercih eden projeler için uygundur:

- Harici bağımlılık yok, kullanıma hazır
- Saf JavaScript geliştirme deneyimi
- Statik web siteleri ve hafif uygulamalar için uygun
- Kademeli işlev genişletme desteği

[HTML sürümü detaylı belgelerini görüntüle](../frameworks/html.md)

### Vue2

Kurumsal uygulama geliştirme için uygundur:

- Tam TypeScript desteği
- Zengin üçüncü parti bileşen ekosistemi
- Olgun geliştirme araç zinciri
- Üretim ortamında kanıtlanmış kararlılık

[Vue2 sürümü detaylı belgelerini görüntüle](../frameworks/vue2.md)

### Vue3

Modern web uygulamaları geliştirme için uygundur:

- Proxy tabanlı reaktif sistem
- Kompozisyon API desteği
- Daha iyi çalışma zamanı performansı
- Daha küçük paket boyutu

[Vue3 sürümü detaylı belgelerini görüntüle](../frameworks/vue3.md)

### Preact+HTM

Hafif ve yüksek performanslı projeler için uygundur:

- Çok küçük çalışma zamanı boyutu (3KB)
- Saf JavaScript şablon sözdizimi
- React ile uyumlu API
- Mükemmel performans

[Preact+HTM sürümü detaylı belgelerini görüntüle](../frameworks/preact-htm.md)

## AI Destekli Geliştirme

Gez Framework, geliştirme verimliliğini önemli ölçüde artırabilen AI destekli geliştirme yetenekleri sunar:

1. Uygun framework sürüm belgelerini seçin
2. Belge içeriğini AI asistanına sağlayın
3. AI otomatik olarak proje iskeleti ve yapılandırma dosyalarını oluşturacaktır

:::tip İpucu
AI destekli geliştirme yalnızca proje başlatma hızını artırmakla kalmaz, aynı zamanda proje yapısının en iyi uygulamalara uygun olmasını sağlar.
:::
