# Início Rápido

## Inicialização do Projeto

Para criar e inicializar um novo projeto Gez, siga os passos abaixo:

```bash
mkdir nome-do-seu-app
cd nome-do-seu-app
```

## Seleção de Versão do Framework

O Framework Gez oferece várias versões de pilha tecnológica, cada uma otimizada para diferentes cenários de aplicação. Escolha a versão adequada de acordo com as necessidades do seu projeto:

### HTML

Ideal para projetos que buscam uma arquitetura minimalista:

- Zero dependências externas, pronto para uso
- Experiência de desenvolvimento com JavaScript nativo
- Adequado para construção de sites estáticos e aplicativos leves
- Suporte para expansão progressiva de funcionalidades

[Ver documentação detalhada da versão HTML](../frameworks/html.md)

### Vue2

Ideal para desenvolvimento de aplicativos empresariais:

- Suporte completo para TypeScript
- Ecossistema rico de componentes de terceiros
- Ferramentas de desenvolvimento maduras
- Estabilidade comprovada em ambientes de produção

[Ver documentação detalhada da versão Vue2](../frameworks/vue2.md)

### Vue3

Ideal para desenvolvimento de aplicativos web modernos:

- Sistema reativo baseado em Proxy
- Suporte para API de composição
- Melhor desempenho em tempo de execução
- Volume de pacote menor

[Ver documentação detalhada da versão Vue3](../frameworks/vue3.md)

### Preact+HTM

Ideal para projetos que buscam leveza e alto desempenho:

- Volume de tempo de execução extremamente pequeno (3KB)
- Sintaxe de modelo JavaScript nativo
- API compatível com React
- Excelente desempenho

[Ver documentação detalhada da versão Preact+HTM](../frameworks/preact-htm.md)

## Desenvolvimento Assistido por IA

O Framework Gez oferece capacidades de desenvolvimento assistido por IA, que podem aumentar significativamente a eficiência do desenvolvimento:

1. Escolha a documentação da versão do framework adequada
2. Forneça o conteúdo da documentação ao assistente de IA
3. A IA gerará automaticamente o esqueleto do projeto e os arquivos de configuração

:::tip Dica
O desenvolvimento assistido por IA não apenas acelera a inicialização do projeto, mas também garante que a estrutura do projeto esteja em conformidade com as melhores práticas.
:::
