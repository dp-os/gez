# Bắt đầu nhanh

## Khởi tạo dự án

Để tạo và khởi tạo một dự án Gez mới, hãy làm theo các bước sau:

```bash
mkdir your-app-name
cd your-app-name
```

## Lựa chọn phiên bản khung

Khung Gez cung cấp nhiều phiên bản công nghệ, mỗi phiên bản được tối ưu hóa cho các tình huống ứng dụng khác nhau. Vui lòng chọn phiên bản phù hợp với nhu cầu dự án của bạn:

### HTML

Phù hợp cho các dự án theo đuổi kiến trúc tối giản:

- Không phụ thuộc bên ngoài, sẵn sàng sử dụng ngay
- Trải nghiệm phát triển JavaScript thuần
- Phù hợp để xây dựng trang web tĩnh và ứng dụng nhẹ
- Hỗ trợ mở rộng tính năng dần dần

[Xem tài liệu chi tiết về phiên bản HTML](../frameworks/html.md)

### Vue2

Phù hợp cho phát triển ứng dụng doanh nghiệp:

- Hỗ trợ TypeScript đầy đủ
- Hệ sinh thái thành phần bên thứ ba phong phú
- Chuỗi công cụ phát triển trưởng thành
- Độ ổn định đã được kiểm chứng trong môi trường sản xuất

[Xem tài liệu chi tiết về phiên bản Vue2](../frameworks/vue2.md)

### Vue3

Phù hợp cho phát triển ứng dụng Web hiện đại:

- Hệ thống phản ứng dựa trên Proxy
- Hỗ trợ API tổ hợp
- Hiệu suất thời gian chạy tốt hơn
- Kích thước gói nhỏ hơn

[Xem tài liệu chi tiết về phiên bản Vue3](../frameworks/vue3.md)

### Preact+HTM

Phù hợp cho các dự án theo đuổi sự nhẹ nhàng và hiệu suất cao:

- Kích thước thời gian chạy cực nhỏ (3KB)
- Cú pháp mẫu JavaScript thuần
- API tương thích với React
- Hiệu suất xuất sắc

[Xem tài liệu chi tiết về phiên bản Preact+HTM](../frameworks/preact-htm.md)

## Phát triển hỗ trợ AI

Khung Gez cung cấp khả năng phát triển hỗ trợ AI, có thể nâng cao đáng kể hiệu quả phát triển:

1. Chọn tài liệu phiên bản khung phù hợp
2. Cung cấp nội dung tài liệu cho trợ lý AI
3. AI sẽ tự động tạo bộ khung dự án và tệp cấu hình

:::tip Mẹo
Phát triển hỗ trợ AI không chỉ tăng tốc độ khởi tạo dự án mà còn đảm bảo cấu trúc dự án tuân thủ các phương pháp tốt nhất.
:::
