# 快速开始

本文将指导您使用 Gez 框架创建一个基于原生 HTML 的服务端渲染（SSR）应用。通过这个示例，您将了解 Gez 的基本用法、项目结构和核心开发流程。本示例适合初学者入门，同时也展示了 Gez 框架的主要特性。

::: tip
你可以全选复制页面内容，并将其发送给 AI 助手，AI 将帮助你快速创建和配置项目。
:::

## 环境要求

### Node.js 版本
需要 Node.js 22.6 或更高版本，这是因为：
- 原生支持 TypeScript 的类型导入（通过 `--experimental-strip-types` 标志）
- 完整支持 ESM 特性，包括动态导入和 import.meta
- 提供更好的开发体验和性能

## 项目初始化

### 1. 创建项目目录
```bash
# 创建并进入项目目录
mkdir your-app-name && cd your-app-name

# 初始化 package.json
npm init -y
```

### 2. 配置项目基础信息
```json title="package.json"
{
    "name": "your-app-name",
    "type": "module",
    "private": true,
    "scripts": {
        "dev": "gez dev",
        "build": "npm run build:dts && npm run build:ssr",
        "build:ssr": "gez build",
        "build:dts": "tsc --declaration --emitDeclarationOnly --outDir dist/src",
        "preview": "gez preview",
        "start": "NODE_ENV=production node dist/index.js"
    }
}
```

### 3. 安装依赖

**运行时依赖**
<PackageManagerTabs command="install @gez/core" />

**开发依赖**
<PackageManagerTabs command="install @gez/rspack typescript @types/node -D" />

::: tip 依赖管理说明
- `@gez/core`：框架核心运行时，提供服务端渲染引擎和模块链接系统，是应用的基础依赖
- `@gez/rspack`：基于 Rspack 的高性能构建工具，支持开发环境的热更新和生产环境的资源优化
- `typescript`：JavaScript 的超集，提供静态类型检查和最新的 ECMAScript 特性支持
- `@types/node`：Node.js 的 TypeScript 类型定义，用于服务端代码的类型提示

:::

### 4. 配置 TypeScript
```json title="tsconfig.json"
{
    "compilerOptions": {
        "module": "ESNext",
        "moduleResolution": "node",
        "isolatedModules": true,
        "resolveJsonModule": true,
        
        "target": "ESNext",
        "lib": ["ESNext", "DOM"],
        
        "strict": true,
        "skipLibCheck": true,
        "types": ["@types/node"],
        
        "experimentalDecorators": true,
        "allowSyntheticDefaultImports": true,
        
        "baseUrl": ".",
        "paths": {
            "your-app-name/src/*": ["./src/*"],
            "your-app-name/*": [".*"]
        }
    },
    "include": ["src"],
    "exclude": ["dist", "node_modules"]
}
```

## 核心代码实现

### 1. 创建项目结构
```
your-app-name/
├── src/
│   ├── entry.client.ts   # 客户端入口：浏览器端逻辑
│   ├── entry.server.ts   # 服务端入口：SSR 渲染逻辑
│   └── entry.node.ts     # 服务器入口：HTTP 服务配置
├── tsconfig.json         # TypeScript 配置
└── package.json          # 项目配置
```

### 2. 配置 HTTP 服务器
```ts title="src/entry.node.ts"
import http from 'node:http';
import type { GezOptions } from '@gez/core';

export default {
    // 配置开发环境的应用创建器
    // 用于设置开发服务器的编译和热更新功能
    async devApp(gez) {
        return import('@gez/rspack').then((m) =>
            m.createRspackHtmlApp(gez, {
                config(context) {
                    // 在此处自定义 Rspack 编译配置
                    // 可配置模块解析、资源处理等选项
                }
            })
        );
    },
    // 配置 HTTP 服务器
    // 处理请求路由和服务端渲染
    async server(gez) {
        const server = http.createServer((req, res) => {
            // 使用 Gez 中间件处理静态资源
            // 支持开发环境的实时编译和生产环境的缓存优化
            gez.middleware(req, res, async () => {
                // 执行服务端渲染
                // 传入 URL 参数用于路由匹配
                const rc = await gez.render({
                    params: {
                        url: req.url
                    }
                });
                // 返回渲染后的 HTML
                res.end(rc.html);
            });
        });
        // 启动 HTTP 服务器并监听 3000 端口
        server.listen(3000, () => {
            console.log('服务器已启动：http://localhost:3000');
        });
    }
} satisfies GezOptions;
```

### 3. 实现服务端渲染
```ts title="src/entry.server.ts"
import type { RenderContext } from '@gez/core';

export default async (rc: RenderContext) => {
    // 提交依赖收集
    await rc.commit();

    // 生成 HTML
    const time = new Date().toISOString();
    rc.html = `
<!DOCTYPE html>
<html>
<head>
    ${rc.preload()}           <!-- 预加载资源 -->
    <title>Gez 示例</title>
    ${rc.css()}               <!-- 注入样式 -->
</head>
<body>
    <h1>Gez 示例</h1>
    <h2>欢迎使用！</h2>
    <p>当前 URL: ${rc.params.url}</p>
    <time>${time}</time>
    ${rc.importmap()}         <!-- 导入映射 -->
    ${rc.moduleEntry()}       <!-- 入口模块 -->
    ${rc.modulePreload()}     <!-- 模块预加载 -->
</body>
</html>
`;
};
```

### 4. 实现客户端交互
```ts title="src/entry.client.ts"
// 实现时间自动更新
const time = document.querySelector('time');
setInterval(() => {
    time?.setHTMLUnsafe(new Date().toISOString());
}, 1000);
```
