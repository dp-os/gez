# 快速开始

## 项目初始化

创建并初始化一个新的 Gez 项目，请按照以下步骤操作：

```bash
mkdir your-app-name
cd your-app-name
```

## 框架版本选择

Gez 框架提供了多个技术栈版本，每个版本都针对不同的应用场景进行了优化。请根据项目需求选择合适的版本：

### HTML

适用于追求极简架构的项目：

- 零外部依赖，启动即用
- 原生 JavaScript 开发体验
- 适合构建静态网站和轻量级应用
- 支持渐进式功能扩展

[查看 HTML 版本详细文档](../frameworks/html.md)

### Vue2

适用于企业级应用开发：

- 完整的 TypeScript 支持
- 丰富的第三方组件生态
- 成熟的开发工具链
- 经过生产环境验证的稳定性

[查看 Vue2 版本详细文档](../frameworks/vue2.md)

### Vue3

适用于现代化 Web 应用开发：

- 基于 Proxy 的响应式系统
- 组合式 API 支持
- 更优的运行时性能
- 更小的打包体积

[查看 Vue3 版本详细文档](../frameworks/vue3.md)

### Preact+HTM

适用于追求轻量级和高性能的项目：

- 极小的运行时体积（3KB）
- 原生 JavaScript 模板语法
- 与 React 兼容的 API
- 优秀的性能表现

[查看 Preact+HTM 版本详细文档](../frameworks/preact-htm.md)

## AI 辅助开发

Gez 框架提供了 AI 辅助开发能力，可以显著提升开发效率：

1. 选择适合的框架版本文档
2. 将文档内容提供给 AI 助手
3. AI 将自动生成项目骨架和配置文件

:::tip 提示
AI 辅助开发不仅能加快项目初始化速度，还能确保项目结构符合最佳实践。
:::
