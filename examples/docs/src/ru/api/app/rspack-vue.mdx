# @gez/rspack-vue

Пакет Rspack Vue предоставляет набор API для создания и настройки приложений на основе Vue Framework с использованием Rspack, поддерживая разработку компонентов Vue, сборку и серверный рендеринг.

## Установка

Установите `@gez/rspack-vue` как зависимость для разработки с помощью менеджера пакетов:

<PackageManagerTabs command="install @gez/rspack-vue -D" />

## Экспорт типов

### BuildTarget

```ts
type BuildTarget = 'node' | 'client' | 'server'
```

Тип целевой среды сборки, определяющий целевую среду для сборки приложения, используется для настройки специфических оптимизаций и функций в процессе сборки:
- `node`: сборка кода для выполнения в среде Node.js
- `client`: сборка кода для выполнения в браузере
- `server`: сборка кода для выполнения на сервере

### RspackAppConfigContext

```ts
interface RspackAppConfigContext {
  gez: Gez
  buildTarget: BuildTarget
  config: RspackOptions
  options: RspackAppOptions
}
```

Интерфейс контекста конфигурации приложения Rspack, предоставляющий информацию, доступную в функциях-хуках конфигурации:
- `gez`: экземпляр Gez Framework
- `buildTarget`: текущая цель сборки (client/server/node)
- `config`: объект конфигурации Rspack
- `options`: опции конфигурации приложения

### RspackAppOptions

```ts
interface RspackAppOptions {
  css?: 'css' | 'style'
  loaders?: {
    styleLoader?: string
  }
  styleLoader?: Record<string, any>
  cssLoader?: Record<string, any>
  target?: {
    web?: string[]
    node?: string[]
  }
  definePlugin?: Record<string, any>
  config?: (context: RspackAppConfigContext) => void | Promise<void>
}
```

Интерфейс опций конфигурации приложения Rspack:
- `css`: способ вывода CSS, возможные значения 'css' (отдельный файл) или 'style' (встроенные стили)
- `loaders`: настройка пользовательских loader'ов
- `styleLoader`: опции конфигурации style-loader
- `cssLoader`: опции конфигурации css-loader
- `target`: настройка совместимости для целевых сред
- `definePlugin`: определение глобальных констант
- `config`: функция-хук конфигурации

### RspackHtmlAppOptions

Наследуется от `RspackAppOptions`, используется для настройки специфических опций HTML-приложений.

## Экспорт функций

### createRspackApp

```ts
function createRspackApp(gez: Gez, options?: RspackAppOptions): Promise<App>
```

Создает стандартный экземпляр приложения Rspack.

**Параметры:**
- `gez`: экземпляр Gez Framework
- `options`: опции конфигурации приложения Rspack

**Возвращаемое значение:**
- Возвращает Promise, который разрешается в созданный экземпляр приложения

### createRspackHtmlApp

```ts
function createRspackHtmlApp(gez: Gez, options?: RspackHtmlAppOptions): Promise<App>
```

Создает экземпляр HTML-приложения Rspack.

**Параметры:**
- `gez`: экземпляр Gez Framework
- `options`: опции конфигурации HTML-приложения

**Возвращаемое значение:**
- Возвращает Promise, который разрешается в созданный экземпляр HTML-приложения

## Экспорт констант
### RSPACK_LOADER

```ts
const RSPACK_LOADER: Record<string, string> = {
  builtinSwcLoader: 'builtin:swc-loader',
  lightningcssLoader: 'builtin:lightningcss-loader',
  styleLoader: 'style-loader',
  cssLoader: 'css-loader',
  lessLoader: 'less-loader',
  styleResourcesLoader: 'style-resources-loader',
  workerRspackLoader: 'worker-rspack-loader'
}
```

Объект сопоставления идентификаторов встроенных loader'ов Rspack, предоставляющий константы для часто используемых loader'ов:

- `builtinSwcLoader`: встроенный SWC loader Rspack для обработки файлов TypeScript/JavaScript
- `lightningcssLoader`: встроенный lightningcss loader Rspack для высокопроизводительной компиляции CSS-файлов
- `styleLoader`: loader для вставки CSS в DOM
- `cssLoader`: loader для разбора CSS-файлов и обработки CSS-модулей
- `lessLoader`: loader для компиляции Less-файлов в CSS
- `styleResourcesLoader`: loader для автоматического импорта глобальных стилевых ресурсов (например, переменных, миксинов)
- `workerRspackLoader`: loader для обработки файлов Web Worker

Использование этих констант позволяет ссылаться на встроенные loader'ы в конфигурации, избегая ручного ввода строк:

```ts title="src/entry.node.ts"
import { RSPACK_LOADER } from '@gez/rspack';

export default {
  async devApp(gez) {
    return import('@gez/rspack').then((m) =>
      m.createRspackHtmlApp(gez, {
        loaders: {
          // Использование констант для ссылки на loader'ы
          styleLoader: RSPACK_LOADER.styleLoader,
          cssLoader: RSPACK_LOADER.cssLoader,
          lightningcssLoader: RSPACK_LOADER.lightningcssLoader
        }
      })
    );
  }
};
```

**Примечания:**
- Эти loader'ы уже встроены в Rspack и не требуют дополнительной установки
- При настройке пользовательских loader'ов можно использовать эти константы для замены реализаций по умолчанию
- Некоторые loader'ы (например, `builtinSwcLoader`) имеют специфические опции конфигурации, обратитесь к соответствующей документации
