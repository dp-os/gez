# Быстрый старт

## Инициализация проекта

Чтобы создать и инициализировать новый проект на Gez, выполните следующие шаги:

```bash
mkdir your-app-name
cd your-app-name
```

## Выбор версии фреймворка

Фреймворк Gez предоставляет несколько версий технологического стека, каждая из которых оптимизирована для различных сценариев использования. Выберите подходящую версию в зависимости от требований вашего проекта:

### HTML

Подходит для проектов, ориентированных на минималистичную архитектуру:

- Нет внешних зависимостей, готов к использованию сразу после установки
- Нативный опыт разработки на JavaScript
- Подходит для создания статических сайтов и легковесных приложений
- Поддержка постепенного расширения функциональности

[Подробная документация по версии HTML](../frameworks/html.md)

### Vue2

Подходит для разработки корпоративных приложений:

- Полная поддержка TypeScript
- Богатая экосистема сторонних компонентов
- Зрелая цепочка инструментов разработки
- Проверенная стабильность в производственной среде

[Подробная документация по версии Vue2](../frameworks/vue2.md)

### Vue3

Подходит для разработки современных веб-приложений:

- Реактивная система на основе Proxy
- Поддержка Composition API
- Улучшенная производительность во время выполнения
- Меньший размер сборки

[Подробная документация по версии Vue3](../frameworks/vue3.md)

### Preact+HTM

Подходит для проектов, ориентированных на минимальный размер и высокую производительность:

- Очень маленький размер во время выполнения (3KB)
- Нативный синтаксис шаблонов JavaScript
- Совместимый с React API
- Отличная производительность

[Подробная документация по версии Preact+HTM](../frameworks/preact-htm.md)

## AI-помощник для разработки

Фреймворк Gez предоставляет возможность использования AI-помощника для разработки, что значительно повышает эффективность:

1. Выберите подходящую версию фреймворка и изучите документацию
2. Передайте содержимое документации AI-помощнику
3. AI автоматически сгенерирует каркас проекта и конфигурационные файлы

:::tip Подсказка
Использование AI-помощника не только ускоряет процесс инициализации проекта, но и гарантирует, что структура проекта соответствует лучшим практикам.
:::
