# Inicio rápido

## Inicialización del proyecto

Para crear e inicializar un nuevo proyecto con Gez, sigue estos pasos:

```bash
mkdir nombre-de-tu-aplicación
cd nombre-de-tu-aplicación
```

## Selección de la versión del framework

El framework Gez ofrece múltiples versiones de pila tecnológica, cada una optimizada para diferentes escenarios de aplicación. Selecciona la versión adecuada según las necesidades de tu proyecto:

### HTML

Ideal para proyectos que buscan una arquitectura minimalista:

- Cero dependencias externas, listo para usar
- Experiencia de desarrollo con JavaScript nativo
- Adecuado para construir sitios web estáticos y aplicaciones ligeras
- Soporte para expansión progresiva de funcionalidades

[Ver documentación detallada de la versión HTML](../frameworks/html.md)

### Vue2

Recomendado para el desarrollo de aplicaciones empresariales:

- Soporte completo para TypeScript
- Ecosistema rico en componentes de terceros
- Cadena de herramientas de desarrollo madura
- Estabilidad probada en entornos de producción

[Ver documentación detallada de la versión Vue2](../frameworks/vue2.md)

### Vue3

Perfecto para el desarrollo de aplicaciones web modernas:

- Sistema reactivo basado en Proxy
- Soporte para API de composición
- Mejor rendimiento en tiempo de ejecución
- Tamaño de paquete más reducido

[Ver documentación detallada de la versión Vue3](../frameworks/vue3.md)

### Preact+HTM

Ideal para proyectos que priorizan la ligereza y el alto rendimiento:

- Tamaño de ejecución extremadamente pequeño (3KB)
- Sintaxis de plantillas en JavaScript nativo
- API compatible con React
- Excelente rendimiento

[Ver documentación detallada de la versión Preact+HTM](../frameworks/preact-htm.md)

## Desarrollo asistido por IA

El framework Gez ofrece capacidades de desarrollo asistido por IA que pueden mejorar significativamente la eficiencia del desarrollo:

1. Selecciona la documentación de la versión del framework adecuada
2. Proporciona el contenido de la documentación al asistente de IA
3. La IA generará automáticamente el esqueleto del proyecto y los archivos de configuración

:::tip Consejo
El desarrollo asistido por IA no solo acelera la inicialización del proyecto, sino que también asegura que la estructura del proyecto siga las mejores prácticas.
:::
