빠른 시작#

프로젝트 초기화#

새로운 Gez 프로젝트를 생성하고 초기화하려면 다음 단계를 따르세요:

1mkdir your-app-name
2cd your-app-name

프레임워크 버전 선택#

Gez 프레임워크는 다양한 기술 스택 버전을 제공하며, 각 버전은 다른 애플리케이션 시나리오에 맞게 최적화되어 있습니다. 프로젝트 요구 사항에 따라 적합한 버전을 선택하세요:

HTML#

극도로 간단한 아키텍처를 추구하는 프로젝트에 적합:

  • 외부 의존성 없이 바로 사용 가능
  • 순수 JavaScript 개발 경험
  • 정적 웹사이트 및 경량 애플리케이션 구축에 적합
  • 점진적 기능 확장 지원

HTML 버전 상세 문서 보기

Vue2#

기업급 애플리케이션 개발에 적합:

  • 완전한 TypeScript 지원
  • 풍부한 서드파티 컴포넌트 생태계
  • 성숙한 개발 도구 체인
  • 생산 환경에서 검증된 안정성

Vue2 버전 상세 문서 보기

Vue3#

현대적인 웹 애플리케이션 개발에 적합:

  • Proxy 기반의 반응형 시스템
  • 컴포지션 API 지원
  • 더 나은 런타임 성능
  • 더 작은 번들 크기

Vue3 버전 상세 문서 보기

Preact+HTM#

경량 및 고성능을 추구하는 프로젝트에 적합:

  • 매우 작은 런타임 크기 (3KB)
  • 순수 JavaScript 템플릿 구문
  • React와 호환되는 API
  • 우수한 성능

AI 지원 개발#

Gez 프레임워크는 AI 지원 개발 기능을 제공하여 개발 효율성을 크게 향상시킬 수 있습니다:

  1. 적합한 프레임워크 버전 문서 선택
  2. 문서 내용을 AI 어시스턴트에 제공
  3. AI가 자동으로 프로젝트 골격과 설정 파일 생성
팁

AI 지원 개발은 프로젝트 초기화 속도를 높일 뿐만 아니라, 프로젝트 구조가 최적의 관행을 따르도록 보장합니다.
