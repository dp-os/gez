ESM 기반 서버 사이드 렌더링 프레임워크

고성능, 확장 가능한 모듈 링크 시스템

빠른 시작
GitHub
⚡️

코어 엔진

ESM Import Maps를 기반으로 한 고성능 SSR 엔진으로, Vue와 React와 같은 주요 프레임워크를 지원하며 완전한 서버 사이드 렌더링 기능을 제공합니다.

🎯

렌더링 컨텍스트

스마트한 RenderContext 시스템으로, 정확한 리소스 주입, 상태 동기화 및 클라이언트 활성화 메커니즘을 제공하여 최적의 렌더링 성능을 보장합니다.

🔗

모듈 링크

혁신적인 모듈 링크 시스템으로, 런타임 동적 로딩을 지원하며 정확한 의존성 관리와 리소스 최적화를 구현합니다.

📦

빌드 시스템

유연한 빌드 설정으로, 다중 버전 출력, 전처리 및 후처리 훅을 지원하며 한 번의 빌드로 여러 곳에 배포할 수 있습니다.

🌍

배포 솔루션

다국어 사이트, 동적 기본 경로 설정, 마이크로 프론트엔드 아키텍처를 지원하며, 동일한 빌드 결과물을 임의의 경로에 배포할 수 있습니다.

🛠️

개발 경험

완전한 TypeScript 지원, 내장된 핫 리로드 및 증분 빌드, 풍부한 디버깅 도구와 성능 최적화 가이드를 제공합니다.

💻

기업 애플리케이션

대형 웹 애플리케이션, 다국어 사이트, 마이크로서비스 아키텍처에 적합하며, 팀 협업과 모듈식 개발을 지원합니다.

🚀

성능 시나리오

전자상거래 웹사이트, 뉴스 미디어와 같은 성능 민감 시나리오에 최적화되어 있으며, SEO 및 초기 로딩 최적화를 지원합니다.

🌐

환경 지원

Node.js 22.9+ 및 최신 브라우저를 지원하며, ESM dynamic import와 import.meta를 완전히 지원합니다.
