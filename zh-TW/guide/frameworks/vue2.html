Vue2#

本教學將幫助你從零開始搭建一個基於 Gez 的 Vue2 SSR 應用。我們將透過一個完整的範例來展示如何使用 Gez 框架建立伺服器端渲染應用。

專案結構#

首先，讓我們了解專案的基本結構：

1.
2├── package.json         # 專案設定檔，定義依賴和指令碼命令
3├── tsconfig.json        # TypeScript 設定檔，設定編譯選項
4└── src                  # 原始碼目錄
5    ├── app.vue          # 主應用元件，定義頁面結構和互動邏輯
6    ├── create-app.ts    # Vue 實例建立工廠，負責初始化應用
7    ├── entry.client.ts  # 客戶端入口檔案，處理瀏覽器端渲染
8    ├── entry.node.ts    # Node.js 伺服器入口檔案，負責開發環境配置和伺服器啟動
9    └── entry.server.ts  # 伺服器端入口檔案，處理 SSR 渲染邏輯

專案配置#

package.json#

建立 package.json 檔案，配置專案依賴和指令碼：

package.json
1{
2  "name": "ssr-demo-vue2",
3  "version": "1.0.0",
4  "type": "module",
5  "private": true,
6  "scripts": {
7    "dev": "gez dev",
8    "build": "npm run build:dts && npm run build:ssr",
9    "build:ssr": "gez build",
10    "preview": "gez preview",
11    "start": "NODE_ENV=production node dist/index.js",
12    "build:dts": "vue-tsc --declaration --emitDeclarationOnly --outDir dist/src"
13  },
14  "dependencies": {
15    "@gez/core": "*"
16  },
17  "devDependencies": {
18    "@gez/rspack-vue": "*",
19    "@types/node": "22.8.6",
20    "typescript": "^5.7.3",
21    "vue": "^2.7.16",
22    "vue-server-renderer": "^2.7.16",
23    "vue-tsc": "^2.1.6"
24  }
25}

建立完 package.json 檔案後，需要安裝專案依賴。你可以使用以下任一命令來安裝：

1pnpm install
2# 或
3yarn install
4# 或
5npm install

這將安裝所有必需的依賴套件，包括 Vue2、TypeScript 和 SSR 相關的依賴。

原始碼結構#

app.vue#

建立主應用元件 src/app.vue，使用 <script setup> 語法：

src/app.vue
1<template>
2    <div id="app">
3        <h1><a href="https://www.jsesm.com/guide/frameworks/vue2.html" target="_blank">Gez 快速開始</a></h1>
4        <time :datetime="time">{{ time }}</time>
5    </div>
6</template>
7
8<script setup lang="ts">
9/**
10 * @file 範例元件
11 * @description 展示一個帶有自動更新時間的頁面標題，用於演示 Gez 框架的基本功能
12 */
13
14import { onMounted, onUnmounted, ref } from 'vue';
15
16// 當前時間，每秒更新一次
17const time = ref(new Date().toISOString());
18let timer: NodeJS.Timeout;
19
20onMounted(() => {
21    timer = setInterval(() => {
22        time.value = new Date().toISOString();
23    }, 1000);
24});
25
26onUnmounted(() => {
27    clearInterval(timer);
28});
29</script>

4 */
5
6import Vue from 'vue';

8

entry.node.ts#

建立 entry.node.ts 檔案，配置開發環境和伺服器啟動：

src/entry.node.ts
1/**
2 * @file Node.js 伺服器入口檔案
3 * @description 負責開發環境配置和伺服器啟動，提供 SSR 執行時環境
4 */
5
6import http from 'node:http';
7import type { GezOptions } from '@gez/core';
8
9export default {
10    /**
11     * 配置開發環境的應用建立器
12     * @description 建立並配置 Rspack 應用實例，用於開發環境的建構和熱更新
13     * @param gez Gez 框架實例，提供核心功能和配置介面
14     * @returns 返回配置好的 Rspack 應用實例，支援 HMR 和即時預覽
15     */
16    async devApp(gez) {
17        return import('@gez/rspack-vue').then((m) =>
18            m.createRspackVue2App(gez, {
19                config(context) {
20                    // 在此處自訂 Rspack 編譯配置
21                }
22            })
23        );
24    },
25
26    /**
27     * 配置並啟動 HTTP 伺服器
28     * @description 建立 HTTP 伺服器實例，整合 Gez 中介軟體，處理 SSR 請求
29     * @param gez Gez 框架實例，提供中介軟體和渲染功能
30     */
31    async server(gez) {
32        const server = http.createServer((req, res) => {
33            // 使用 Gez 中介軟體處理請求
34            gez.middleware(req, res, async () => {
35                // 執行伺服器端渲染
36                const rc = await gez.render({
37                    params: { url: req.url }
38                });
39                res.end(rc.html);
40            });
41        });
42
43        server.listen(3000, () => {
44            console.log('服務啟動: http://localhost:3000');
45        });
46    }
47} satisfies GezOptions;

這個檔案是開發環境配置和伺服器啟動的入口檔案，主要包含兩個核心功能：

  1. devApp 函式：負責建立和配置開發環境的 Rspack 應用實例，支援熱更新和即時預覽功能。這裡使用 createRspackVue2App 來建立專門用於 Vue2 的 Rspack 應用實例。

  2. server 函式：負責建立和配置 HTTP 伺服器，整合 Gez 中介軟體處理 SSR 請求。

entry.server.ts#

建立伺服器端渲染入口檔案 src/entry.server.ts：

src/entry.server.ts
1/**
2 * @file 伺服器端渲染入口檔案
3 * @description 負責伺服器端渲染流程、HTML 生成和資源注入
4 */
5
6import type { RenderContext } from '@gez/core';
7import { createRenderer } from 'vue-server-renderer';
8import { createApp } from './create-app';
9
10// 建立渲染器
11const renderer = createRenderer();
12
13export default async (rc: RenderContext) => {
14    // 建立 Vue 應用實例
15    const { app } = createApp();
16
17    // 使用 Vue 的 renderToString 生成頁面內容
18    const html = await renderer.renderToString(app, {
19        importMetaSet: rc.importMetaSet
20    });
21
22    // 提交依賴收集，確保所有必要資源都被載入
23    await rc.commit();
24
25    // 生成完整的 HTML 結構
26    rc.html = `<!DOCTYPE html>
27<html lang="zh-TW">
28<head>
29    ${rc.preload()}
30    <title>Gez 快速開始</title>
31    ${rc.css()}
32</head>
33<body>
34    ${html}
35    ${rc.importmap()}
36    ${rc.moduleEntry()}
37    ${rc.modulePreload()}
38</body>
39</html>
40`;
41};

執行專案#

完成上述檔案配置後，你可以使用以下命令來執行專案：

  1. 開發模式：
1npm run dev
  1. 建構專案：
1npm run build
  1. 生產環境執行：
1npm run start

現在，你已經成功建立了一個基於 Gez 的 Vue2 SSR 應用！訪問 http://localhost:3000 即可看到效果。
